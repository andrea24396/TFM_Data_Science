Normalización imágenes¶

Crearemos una funcin que normalice las imgenes. Para ello, buscar el valor mximo de cada imagen, y la dividir toda ella por este valor. De este modo, el mapa de pxeles que compondr la imagen tendr un rango de 0 a 1.

Asmismo, esta funcin no har ninguna accin sobre las imgenes que sean totalmente negras, es decir, las que se componen exclusivamente por 0s. Estas las dejar intactas, puesto que si las dividiera por su mximo (en este caso, 0), seran matrices compuestas por nan. De este modo, las imgenes seguirn siendo mapas de 0s.

In [43]:
def normalizar(imagenes):
    """Normaliza las imgenes contenidas en el array que se pasa como parmetro,
    dividindo el valor de todos sus pxeles por el mximo.
    No hace nada si la imagen es totalmente negra, i.e. suma = 0.
    De este modo el mapa de pxeles tendr valores entre 0 y 1"""
    imgs = imagenes.copy()
    
    imgs_norm = [imgs[i]/np.max(imgs[i]) if np.sum(imgs[i])!=0 else imgs[i] for i in range(len(imgs)) ]  
    imgs_norm = np.asarray(imgs_norm)
        
    return imgs_norm

Eliminar imágenes negras¶

Las primeras y ltimas imgenes de cada escner de RM son totalmente negras, por lo que no aportan ninguna informacin til a nuestra red. Es por esto que las eliminaremos para reducir las dimensiones de nuestro dataset y que el entrenamiento de la red neuronal no se demore demasiado.

Tambin es conveniente eliminar aquellas imgenes que son prcticamente negras en su totalidad y tampoco aportan mucha info para el entrenamiento. Estas imgenes tienen pxeles distintos de 0, pero estableceremos un umbral (suma de los pxeles de la imagen) mnimo a partir del cual consideraremos que la imagen aporta informacin relevante a nuestra red.

In [47]:
def eliminar_no_relevantes(imagenes, etiquetas, umbral): 
    """Del array de imgenes de rebanadas de cerebro que se pasa, guarda el ndice de aquellas 
    cuya suma sea inferior a un umbral, ya que sern imgenes casi completamente negras que no aporten mucha informacin. 
    Se obtienen los ndices de las imgenes que nos interesan y se hace slicing del array original con estos ndices.
    Hace lo mismo con las etiquetas correspondientes."""
    
    # posiciones de las imgenes casi enteras negras
    indices_negras = [i for i,img in enumerate(imagenes) if np.sum(imagenes[i]) < umbral]
    indices_negras = np.asarray(indices_negras) # convertimos la lista a array
    
    indices_todas = np.arange(len(imagenes))
    
    # posiciones de las imgenes con las que nos queremos quedar
    ind_buenas = np.setdiff1d(indices_todas, indices_negras)
    
    imagenes2 = imagenes[ind_buenas]
    etiquetas2 = etiquetas[ind_buenas]
    
    return imagenes2, etiquetas2

Binarizar etiquetas¶

Las etiquetas proporcionada por la organizacin de BraTS 2018 muestran la segmentacin del glioma en el cerebro, diferenciando varias partes.

Para hacerlo ms sencillo, se ha decidido convertir estas etiquetas en binarias, de modo que las imgenes segmentadas sean mapas de 0s y 1s. De este modo, simplemente se identificara la posicin del tumor, sin hacer ninguna distincin de sus partes

In [64]:
# FUNCIN PARA HACER ETIQUETAS BINARIAS
# En agradecimientos, incluir a Stack overflow, por salvarnos la vida con las dudas de cdigo

def binarizar_etiqueta(etiquetas):
    """Funcin a la que se le pasa un array de imgenes segmentadas. 
    Hace una copia para que no modificar el array original.
    Convierte el mapa de pxeles en binario, haciendo 1 cualquier pxel con valor distinto de 0."""
    
    tags = etiquetas.copy()
    for tag in tags:
        tag[tag > 0] = 1
    
    return tags
